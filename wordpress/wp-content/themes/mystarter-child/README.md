# MyStarter Child

Thème enfant de base pour le projet courant. Utilise le thème parent **MyStarter**.

## Utilisation

1. Activer le thème *MyStarter Child* dans l’admin WordPress.
2. Ajouter les surcharges spécifiques au client :
   - CSS dans `style.css`
   - PHP (hooks, CPT additionnels…) dans `functions.php`
   - Templates ou patterns dans des sous-dossiers dédiés si besoin

Le parent MyStarter apporte déjà la librairie de blocs personnalisés et les styles globaux.
